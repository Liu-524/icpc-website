Greatest common divisor $GCD(a,b)$ of two positive integers <span class="tex-span"><i>a</i></span> and <span class="tex-span"><i>b</i></span> is equal to the biggest integer <span class="tex-span"><i>d</i></span> such that both integers <span class="tex-span"><i>a</i></span> and <span class="tex-span"><i>b</i></span> are divisible by <span class="tex-span"><i>d</i></span>. There are many efficient algorithms to find greatest common divisor $GCD(a,b)$, for example, Euclid algorithm. <br>Formally, find the biggest integer <span class="tex-span"><i>d</i></span>, such that all integers $a, a+1, a+2, ..., b$ are divisible by <span class="tex-span"><i>d</i></span>. To make the problem even more complicated we allow <span class="tex-span"><i>a</i></span> and <span class="tex-span"><i>b</i></span> to be up to googol, <span class="tex-span">10<sup class="upper-index">100</sup></span> — such number do not fit even in 64-bit integer type!<br>
## Input: 
The only line of the input contains two integers <span class="tex-span"><i>a</i></span> and <span class="tex-span"><i>b</i></span> ($1 \\le a \\le b \\le 10^{100}$).<br>
## Output: 
Output one integer — greatest common divisor of all integers from <span class="tex-span"><i>a</i></span> to <span class="tex-span"><i>b</i></span> inclusive.<br>
