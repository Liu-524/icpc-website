# ACM-ICPC Website
This is the website repository for the ICPC team at UIUC.

## Quick Start
<pre><code>$ git clone https://github.com/acm-uiuc/icpc-website.git
$ cd icpc-website
$ jekyll serve --watch
$ open http://localhost:4000/</code></pre>
